# SkillSphere – Online Learning Platform

A modern online learning platform where users can explore courses, watch lessons, and enroll in skill-based programs like Web Development, Design, Marketing, and more.

## 🔗 Live URL
[Click here to view the project](https://skillshare-mongo-db-kf3.vercel.app/)

## ✨ Key Features

- Browse and search all courses by title
- View top-rated and trending courses on the home page
- Protected course details page (login required)
- Email & password authentication via Better Auth
- Google OAuth sign-in
- User profile page with name and avatar update
- Fully responsive on mobile, tablet, and desktop
- Toast notifications for all user actions
- Custom 404 not-found page

## 📦 NPM Packages Used

| Package | Purpose |
|---|---|
| `better-auth` | Authentication (email/password + Google OAuth) |
| `mongodb` | Database adapter for Better Auth |
| `framer-motion` | Animations on the hero banner |
| `swiper` | (installed, available for slider use) |
| `react-hot-toast` | Toast notifications |
| `react-icons` | UI icons throughout the app |
| `next` | React framework with App Router |
| `tailwindcss` | Utility-first CSS styling |
| `daisyui` | Tailwind component library |
